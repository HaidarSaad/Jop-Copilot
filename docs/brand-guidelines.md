## Voice & Tone

### Voice Attributes

| Attribute | Description | Example |
|-----------|-------------|---------|
| **Practical** | Focus on utility, not hype | "Generate a tailored CV in 30 seconds" |
| **Transparent** | Honest about AI limits | "AI rewrites bullets — you verify facts" |
| **Empowering** | User stays in control | "Your data, your keys, your device" |
| **Concise** | Respect user's time | Short sentences, active voice |

### Tone by Context

| Context | Tone |
|---------|------|
| Landing/Hero | Confident, inspiring |
| Wizard Steps | Instructional, encouraging |
| Errors | Clear, actionable, no blame |
| Settings | Technical, precise |
| Success | Warm, celebratory |

### Writing Rules

- **No buzzwords**: Avoid "orchestrate", "leverage", "synergy", "revolutionize"
- **No placeholders**: Never use "[Insert Name]", "[Company]" — use real examples
- **Active voice**: "AI generates your CV" not "Your CV is generated by AI"
- **Specific > vague**: "30 req/min free tier" not "generous free tier"
- **English-only output**: All AI-generated content in English

Minimum clear space = height of "J" in "Job" on all sides.

### Minimum Size

- Wordmark: 120px width
- Icon: 32px × 32px

### Forbidden

- ❌ Recoloring outside brand palette
- ❌ Rotation, distortion, effects
- ❌ Placement on busy backgrounds without overlay
- ❌ Using icon alone without "AI Job Copilot" nearby (first use)

## Component Standards

### Buttons

| Variant | Background | Text | Border | Usage |
|---------|------------|------|--------|-------|
| Primary | Accent gradient | White | None | Main CTAs |
| Secondary | Primary | White | None | Section CTAs |
| Outline | Transparent | Primary | 2px Primary | Alternate actions |
| Ghost | Transparent | Primary | None | Tertiary actions |
| Destructive | Destructive | White | None | Delete, remove |

**States**: Hover (opacity 0.9), Active (scale 0.98), Focus (ring), Disabled (opacity 0.5)

### Inputs

- Height: `44px` (mobile) / `40px` (desktop)
- Padding: `12px 16px`
- Border: `1px solid var(--color-card-border)`
- Focus: `2px solid var(--color-ring)` + ring offset
- Radius: `var(--radius-lg)` (`12px`)

### Cards

- Background: `var(--color-card)`
- Border: `1px solid var(--color-card-border)`
- Radius: `var(--radius-xl)` (`16px`)
- Shadow: `var(--shadow-sm)` → `var(--shadow-lg)` on hover
- Padding: `24px`

### Toasts

- Radius: `var(--radius-xl)`
- Padding: `16px`
- Shadow: `var(--shadow-lg)`
- Variants: Default, Success (accent border), Destructive

## Dark Mode

- **Strategy**: CSS custom properties + `.dark` class on `<html>`
- **Persistence**: `localStorage.jc_dark_mode` + inline script in `<head>` (pre-hydration)
- **Transition**: `transition-colors` on all themed elements
- **Images**: No dark-mode specific images (use CSS filters if needed)

---

## Accessibility

- **Contrast**: WCAG AA minimum (4.5:1 text, 3:1 UI)
- **Focus**: Visible ring on all interactive elements
- **Motion**: `prefers-reduced-motion` respected (animations → 0.01ms)
- **Keyboard**: Full navigation, skip links where needed
- **Screen readers**: Semantic HTML, ARIA labels on icon-only buttons

## Approval Checklist

Before shipping any branded asset:

- [ ] Colors match palette exactly (no eyedropper approximations)
- [ ] Typography uses Space Grotesk/Inter/JetBrains Mono only
- [ ] Spacing follows 4px base unit
- [ ] Radius uses defined scale
- [ ] Shadows use defined scale
- [ ] Dark mode tested
- [ ] Reduced motion tested
- [ ] Focus states visible
- [ ] Contrast verified (light + dark)
- [ ] Logo clear space respected
- [ ] No emojis as structural icons
- [ ] Voice matches context (see Tone by Context)
