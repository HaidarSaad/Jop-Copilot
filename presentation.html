🎨 Design System

CSS variables + Tailwind 4 + Inter font + Phosphor Icons. Dark mode via CSS custom properties with inline script (no flash).

🧩 shadcn/ui Components

Button, Input, Textarea, Card, Toast with Radix UI primitives. Full focus-visible, reduced-motion, keyboard navigation support.

Professional PDF Output

Not just text — publication-ready CVs with proper formatting, smart text sanitization, and branded filenames

Smart Formatting

  • ✓ Bold parsing: Company Name → PDF bold
  • ✓ Section headers: ## **Experience** with --- separators
  • ✓ Bullet preservation: - Achievement → proper bullets
  • ✓ Arabic/RTL text support with proper alignment
  • ✓ Hyphenation repair for copy-pasted vertical text
  • ✓ [ADD]/[/ADD] green highlights stripped on download

Filename Convention

{CandidateName}-{JobTitle}-{Suffix}.pdf

Examples:
Sarah Chen-Senior Engineer-CV.pdf
Ahmed Hassan-Product Manager-Tailored CV.pdf
Maria Santos-Marketing Lead-Cover Letter.pdf

Extract Logic

Tailored CV download strips the ## Tailored CV header and description line automatically — raw stored value retains both sections.

Data Flow Architecture

Client-only, privacy-first, zero server dependencies

localStorage (jc_*) → Wizard State → Prompt Builder → Sanitize → Provider API → Result → Storage → PDF/Clipboard
